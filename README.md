# Algoritmos e Lógica de Programação (Java)
 
Coletânea de exercícios resolvidos do curso de **Algoritmos e Lógica de Programação** do Prof. Dr. Nelio Alves, desenvolvidos em **Java**.
 
## Sobre
 
Repositório usado para praticar e organizar os exercícios feitos ao longo do curso, acompanhando o conteúdo de cada seção.

## Conceitos praticados
 
- Lógica de programação
- Estruturas sequenciais, condicionais e repetitivas
- Vetores e matrizes
- Entrada e saída de dados
## Tecnologias
 
- Java
